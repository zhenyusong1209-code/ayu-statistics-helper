# 网页版使用说明

## 概述

号码统计助手已优化为响应式网页版，支持手机、平板和PC端访问，界面会自动适应不同屏幕尺寸。

---

## 访问方式

### 开发环境

```bash
# 启动开发服务器
pnpm dev:web

# 访问地址
http://localhost:5000
```

### 生产环境

```bash
# 编译生产版本
pnpm build:web

# 编译后的文件在 dist-web 目录
```

部署时，将 `dist-web` 目录下的所有文件上传到您的Web服务器即可。

---

## 响应式设计

### 移动端（< 640px）
- 全宽显示，充分利用屏幕空间
- 标签栏横向滚动，紧凑布局
- 触摸友好的按钮和交互

### 平板端（640px - 1024px）
- 内容区域限制最大宽度，居中显示
- 保持良好的触摸体验
- 优化间距和字体大小

### PC端（> 1024px）
- 内容区域最大宽度 896px (max-w-4xl)
- 居中显示，两侧留白
- 优化鼠标交互体验
- 更大的点击区域

## 功能模块

### 1. 挑码
- 根据条件筛选号码
- 支持按生肖、波色、五行等条件筛选
- 实时显示筛选结果

### 2. 统计
- 统计号码出现次数
- 按出现次数分组显示
- 支持号码和生肖统计

### 3. 复式
- 支持复式计算（复2、复3、复4、复5）
- 自动生成所有组合
- 支持数字和生肖两种模式

### 4. 选号
- 智能选号功能
- 按生肖分组显示
- 支持点击生肖查看/隐藏号码
- 一键复制选号结果

### 5. 关于我们
- 应用介绍
- 版本信息

## 技术特性

### 响应式框架
- **Tailwind CSS 4.x**: 原子化CSS，快速构建响应式界面
- **Grid 布局**: 生肖卡片使用 `grid-cols-12` 自适应排列
- **Flex 布局**: 内容区域使用 flexbox 实现灵活布局

### 断点设置
```css
sm:  640px  /* 小平板 */
md:  768px  /* 平板 */
lg:  1024px /* 小屏幕笔记本 */
xl:  1280px /* 桌面 */
```

### 最大宽度限制
```css
max-w-4xl = 56rem = 896px
```

## 部署建议

### 静态托管
- **GitHub Pages**: 免费，适合开源项目
- **Netlify**: 免费，支持自动部署
- **Vercel**: 免费，性能优秀
- **阿里云OSS**: 国内访问速度快

### 部署步骤（以 Netlify 为例）

1. 注册 Netlify 账号
2. 连接 Git 仓库或拖拽上传 `dist-web` 目录
3. 配置构建命令: `pnpm build:web`
4. 设置发布目录: `dist-web`
5. 点击部署

## 问题排查

### Q: 页面空白？
**A:** 检查浏览器控制台是否有错误，确认 JavaScript 已启用。

### Q: 样式不正确？
**A:** 清除浏览器缓存，重新刷新页面。

### Q: 无法在手机上访问？
**A:** 确保手机和电脑在同一局域网，使用电脑的 IP 地址访问。
