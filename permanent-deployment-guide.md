## 🇨🇳 方案 2: 腾讯云/阿里云（国内访问最快）

### 优势
- ✅ 国内访问速度快
- ✅ 域名便宜
- ✅ 服务稳定
- ✅ 有中文支持

### 费用
- **域名**: 9-69 元/年
- **服务器**: 50-100 元/年（轻量应用服务器）
- **总计**: 约 60-170 元/年

### 部署步骤
1. **购买域名**
   - 腾讯云：https://dnspod.cloud.tencent.com/
   - 阿里云：https://wanwang.aliyun.com/

2. **购买服务器**
   - 腾讯云轻量应用服务器（50元/年）
   - 阿里云云服务器ECS（89元/年）

3. **配置服务器**
   ```bash
   # 安装 Nginx
   sudo apt update
   sudo apt install nginx

   # 上传构建文件到 /var/www/html
   scp -r dist-web/* root@your-server:/var/www/html/
   ```

4. **配置 HTTPS**
   - 使用 Let's Encrypt 免费证书
   - 或使用云服务商提供的证书

### 预计成本
- **60-170 元/年**

## 🚀 快速开始（推荐 Vercel）

### 1. 运行准备脚本
```bash
./deploy-vercel.sh
```

### 2. 创建 GitHub 仓库
访问：https://github.com/new

### 3. 推送代码
```bash
git add .
git commit -m "feat: 初始化项目"
git remote add origin https://github.com/你的用户名/ayu-statistics-helper.git
git push -u origin main
```

### 4. Vercel 部署
1. 访问 https://vercel.com/dashboard
2. 点击 "Add New" → "Project"
3. 选择你的仓库
4. 点击 "Deploy"

### 5. 获取永久域名
```
https://ayu-statistics-helper.vercel.app
```
